Tags Galore

In the Tags Galore Activity project, we learned how to create an HTML file and utilize various HTML elements. We explored headings (H1, H2, H3) and paragraph text, and incorporated formatting tags like bold and italics. After reviewing familiar and new HTML elements on W3 Schools, we selected five tags to include in our project. We then accepted a GitHub Classroom assignment, cloned the repository, created a new HTML file, added the selected elements, and pushed our changes to the remote repo. Finally, we submitted the URL of our GitHub repository for the assignment.

CSS Properties

In the CSS Properties Activity project, we practiced our CSS skills by creating an HTML document named "index.html" with at least five elements and linking it to an external CSS document named "styles.css." The CSS file included at least 15 unique properties to style the HTML elements. We followed steps to accept the GitHub Classroom assignment, cloned the repository, and worked within the cloned repo. After completing the assignment, we pushed our changes to GitHub and submitted the link to our repository for evaluation.

My Favorite Recipe

In the final part of the recipe site project, we focused on responsive design to ensure the website looks good on desktops and mobile devices. We used CSS media queries for different screen sizes, ensured images scaled correctly, tested the site on multiple devices and browsers, and optimized loading times and performance. This followed Parts 1 and 2, where we created HTML files with basic and advanced CSS styling, including a multi-page layout with text, images, lists, and tables. We then uploaded the complete project to GitHub and submitted the repository link for grading.
